ПОДКЛЮЧЕНИЕ К СУПЕРКОМПЬЮТЕРУ "ПОЛИТЕХНИК"
===========================================

ШАГ 1: СОЗДАНИЕ SSH-КЛЮЧЕЙ
----------------------------

1. Скачайте PuTTYgen:
   https://www.putty.org/ → Download → PuTTYgen

2. Запустите PuTTYgen.exe

3. Настройки:
   - Тип ключа: SSH-2 RSA
   - Размер: 2048 бит (или 4096, если доступно)

4. Нажмите "Generate"

5. Поводите мышкой в пустой области до заполнения шкалы прогресса

6. Введите пароль (запомните его!):
   - Key passphrase: введите пароль
   - Confirm passphrase: повторите пароль

7. Сохраните ключи:
   - Нажмите "Save public key" → сохраните как: id_rsa.pub
   - Нажмите "Save private key" → сохраните как: mykey.ppk
     (при сохранении будет предупреждение - нажмите "Да")

8. Скопируйте содержимое открытого ключа:
   - Выделите весь текст в поле "Public key for pasting into OpenSSH authorized_keys file"
   - Скопируйте (Ctrl+C)

9. Отправьте открытый ключ в службу регистрации СКЦ:
   - Отправьте файл id_rsa.pub или скопированный текст
   - Дождитесь подтверждения регистрации


ШАГ 2: ПОДКЛЮЧЕНИЕ ЧЕРЕЗ PuTTY
-------------------------------

1. Скачайте PuTTY:
   https://www.putty.org/ → Download → PuTTY

2. Запустите PuTTY.exe

3. Настройки подключения:
   - Host Name (or IP address): login1.hpc.spbstu.ru
   - Port: 22
   - Connection type: SSH

4. Настройка SSH ключа:
   - В левом меню: Connection → SSH → Auth
   - В поле "Private key file for authentication" нажмите "Browse"
   - Выберите файл mykey.ppk (который сохранили ранее)

5. Сохранение сессии (опционально, но удобно):
   - Вернитесь в Session
   - В поле "Saved Sessions" введите: Политехник
   - Нажмите "Save"

6. Подключение:
   - Нажмите "Open"
   - В появившемся окне введите ваш логин
   - Нажмите Enter
   - Введите пароль от ключа (который задавали при создании)
   - Нажмите Enter

7. Если все правильно - увидите приглашение командной строки:
   [ваш_логин@login1 ~]$


ШАГ 3: КОПИРОВАНИЕ ФАЙЛОВ ЧЕРЕЗ WinSCP
---------------------------------------

1. Скачайте WinSCP:
   https://winscp.net/eng/download.php

2. Запустите WinSCP.exe

3. Настройки подключения:
   - File protocol: SFTP
   - Host name: login1.hpc.spbstu.ru
   - Port number: 22
   - User name: ваш_логин
   - Password: оставьте пустым (будет использован ключ)

4. Настройка ключа:
   - Нажмите "Advanced..."
   - Перейдите: SSH → Authentication
   - В поле "Private key file" нажмите "..."
   - Выберите файл mykey.ppk
   - Нажмите "OK"

5. Сохранение сессии:
   - Нажмите "Save"
   - Введите имя: Политехник
   - Нажмите "OK"

6. Подключение:
   - Выберите сохраненную сессию "Политехник"
   - Нажмите "Login"
   - Введите пароль от ключа
   - Нажмите "OK"

7. После подключения:
   - Слева: ваш компьютер
   - Справа: суперкомпьютер
   - Перетаскивайте файлы между окнами


ШАГ 4: ПЕРВЫЕ ШАГИ НА СУПЕРКОМПЬЮТЕРЕ
--------------------------------------

После подключения через PuTTY выполните:

   # Проверьте где вы находитесь
   pwd

   # Посмотрите содержимое текущей директории
   ls -la

   # Перейдите в домашнюю директорию
   cd ~

   # Создайте рабочую директорию
   mkdir work
   cd work

   # Клонируйте ваш репозиторий
   git clone https://github.com/ваш_username/ваш_репозиторий.git
   cd ваш_репозиторий

   # Проверьте файлы
   ls -la


ВАЖНЫЕ КОМАНДЫ:
---------------
pwd                    # Показать текущую директорию
ls -la                 # Список файлов
cd ~                   # Перейти в домашнюю директорию
cd директория          # Перейти в директорию
mkdir имя              # Создать директорию
rm файл                # Удалить файл
cat файл               # Показать содержимое файла
nano файл              # Редактировать файл (Ctrl+X для выхода)


ЕСЛИ НЕ ПОДКЛЮЧАЕТСЯ:
---------------------
1. Проверьте правильность адреса: login1.hpc.spbstu.ru
2. Проверьте что ключ зарегистрирован в СКЦ
3. Проверьте что используете правильный файл mykey.ppk
4. Проверьте что вводите правильный пароль от ключа
5. Попробуйте подключиться без ключа (только логин/пароль) для проверки
